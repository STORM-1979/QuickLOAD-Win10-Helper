# QuickLOAD Windows 10/11 Helper

Вспомогательные скрипты и языковые файлы для запуска QuickLOAD/QuickTARGET (H.G. Broemel, v3.8) на Windows 10/11.

## Проблема

QuickLOAD 3.8 — 32-битное VB5-приложение 2013 года. На Windows 10/11 возникают три проблемы:

1. **Штатный установщик падает** с ошибкой `Path/File access error` — особенно если активен Avast (он блокирует VB5-инсталляторы как подозрительные)
2. **MSCOMCTL.OCX** и другие VB6-компоненты не зарегистрированы в системе по умолчанию
3. **Ошибка `Pos. 10 — Program installation not complete`** — машинно-зависимая проверка лицензии требует прохождения через оригинальный `setupql8.exe` до шага персонализации

## Решение

1. **Отключить Avast перед установкой** (временно)
2. Установить штатный `SETUP.EXE` с смонтированного ISO от имени администратора
3. В установщике выбрать любой путь (например `C:\QuickLOAD`), заполнить персонализацию (любое имя/организация)
4. Дождаться завершения установки (возможны диалоги про перезапись файлов — «Нет» = не отменять, продолжить)
5. Программа будет готова к запуску из `C:\Program Files (x86)\QuickLOAD\QLOADFW.EXE`

После установки Avast можно включить обратно. Рекомендуется добавить `QLOADFW.EXE` в исключения антивируса.

## Русский язык

Русские языковые файлы не входят в оригинальный ISO. Файлы из папки [lan_russian/](lan_russian/) скопировать в:

```
C:\Users\<YourName>\Documents\QuickLOAD\lan\
```

Затем в программе: **File → Set language to → Russian**

### Правки кодировки в русских файлах

- `charset=255` (OEM/DOS CP866) заменён на `charset=204` (RUSSIAN_CHARSET/CP1251) — без этого VB5-контролы рендерили Кириллицу как кракозябры Latin-1
- Warning-баннер и подписи осей графика переведены в английский вариант — эти VB5-контролы имеют захардкоженный font charset в EXE, без реверс-инжиниринга исправить нельзя
- Длинные русские подписи формы сокращены чтобы помещаться в исходные English-размеры полей

## Французский язык

В оригинальном ISO французских `.lan`/`.tip` файлов нет — только PDF-руководства. В папке [lan_french/](lan_french/) — файлы, созданные на базе английских с частичным переводом ключевых UI-терминов (меню, кнопки, основные поля).

Скопировать в:
```
C:\Users\<YourName>\Documents\QuickLOAD\lan\
```

Затем: **File → Set language to → French**

Большая часть интерфейса останется на английском (т.к. это базовые файлы с поверхностным переводом), но меню и основные элементы переведены.

## База данных (пули / пороха / калибры)

QuickLOAD 3.8 поставляется с базой 2014 года. H.G. Broemel регулярно выпускает обновления базы для 3.6-3.9 — актуальные пули, пороха, калибры.

### Как применить официальное обновление (не включено в репо — платное)

1. Купить на [quickload.co.uk](https://quickload.co.uk/) / [neconos.com](https://www.neconos.com/) / [jmssporting.com](https://www.jmssporting.com/) — от £30 за скачивание (активно 14 дней) или CD
2. Распаковать архив
3. Скопировать содержимое `data\bullets\`, `data\powders\`, `data\calibers\` (и по желанию `data\pictures\`) поверх существующих в:
   ```
   C:\Users\<YourName>\Documents\QuickLOAD\data\
   ```
4. Обновлённый `nameprim.txt` скопировать в ту же папку `Documents\QuickLOAD\`
5. Перезапустить QuickLOAD — админ-права не требуются

### Версии обновлений

- **Февраль 2019** (v3.9 update) — последнее "народное" обновление, часто есть в архивах у пользователей
- **4 октября 2025** — последнее официальное, Windows 11 compliant, содержит данные 2019-2025 гг (новые калибры 6 mm ARC, 277 Fury, .300 PRC, новые Vihtavuori/Reloder, Hornady ELD-X и т.д.)

## Скрипты

- [`START.cmd`](START.cmd) — одноразовый автонастройщик: запускается от админа, регистрирует OCX, монтирует ISO, запускает SETUP.EXE
- [`START.ps1`](START.ps1) — PowerShell-бэкенд для START.cmd
